# Description
- This is an interactive claw machine game where users can control the claw and move it to grab toys and drop them to collect them.

# Features
- Controls
- Different toy types
- Movements
- Animated/Visual effects

# Why I made this
- To experiment with game mechanics like moving the claw and grabbing the toy
- To create something new, fun and cute

# Tools
- HTML
- CSS
- JavaScript

# Learned/Challenges
- Applied CSS animation and transitions to create movements
- Handling multiple toys with animations
